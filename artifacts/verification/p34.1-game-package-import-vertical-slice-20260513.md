## 2. API Endpoint'leri

### POST /internal/game-packages/validate

Valid FRUIT_SLASH:
```json
{"valid":true,"errors":[],"warnings":[],"summary":{"title":"...","template":"FRUIT_SLASH",...},"runtimeCompatibility":{"templateSupported":true,...}}
```

Invalid (BOSS_CHALLENGE + local://):
```json
{"valid":false,"errors":[{"code":"unsupported_template",...},{"code":"local_uri_not_allowed",...}]}
```

Missing cover:
```json
{"valid":false,"errors":[{"code":"missing_cover","message":"Kapak görseli eksik..."}]}
```

### POST /internal/game-packages/import

```json
{"imported":true,"gameDefinitionId":"game_...","status":"DRAFT","redirectTo":"/games/game_.../studio"}
```

## 3. Validation Kuralları

| Kural | Kod | Severity |
|---|---|---|
| schemaVersion != "1.0" | invalid_schema_version | ERROR |
| packageType != "ALBAGO_GAME_PACKAGE" | invalid_package_type | ERROR |
| game.title yok | missing_game_title | ERROR |
| Desteklenmeyen template | unsupported_template | ERROR |
| Geçersiz kategori | invalid_category | ERROR |
| Geçersiz orientation | invalid_orientation | ERROR |
| Geçersiz camera | invalid_camera_requirement | ERROR |
| Süre < 5 veya > 600 | invalid_duration | ERROR |
| Cover yok | missing_cover | ERROR |
| local:// URI | local_uri_not_allowed | ERROR |
| Desteklenmeyen motion | unsupported_motion | ERROR |
| Yasaklı alan (token, secret, script, vb) | forbidden_field | ERROR |
| Private IP / localhost URL | invalid_asset_url | ERROR |
