## 3. Uctan Uca Zincir

```
Admin panel (localhost:3001)
  -> POST /internal/assets (backend proxy)
    -> PHP POST /api/assets/upload (Bearer token, server-side)
      -> asset-manifest.json + backup + upload-log.jsonl
      -> Public URL: https://albago.tr/assets/covers/test.png

Admin Asset Library (/assets)
  -> GET /internal/assets/list
    -> PHP GET /api/assets
  <- Thumbnail grid (remote URL <img>)

Game Studio (/games/[id]/studio)
  -> ASSETS tab -> "Asset Sec"
    -> AssetPickerModal -> listAssets({ category: 'covers' })
    -> onSelect -> draft.assets.cover = asset.url

Publish
  -> validateGameDefinition -> validateCoverAsset
    -> cover missing/local/file/private-ip -> ERROR (gate)
    -> valid HTTPS -> PASS
  -> PATCH /internal/game-definitions/:id -> assets.cover JSON

Active Endpoint
  -> GET /v1/game-definitions/active?appVersion=0.2.0
  <- items[].assets.cover = "https://albago.tr/assets/covers/..."

Android
  -> getActiveGames() -> metadata["assets"]["cover"]
  -> AssetManifest.cover -> GameDefinition.assets.cover
  -> GameCardUi(imageUrl) -> Coil AsyncImage -> Remote cover
  -> Fallback: painterResource(R.drawable.demo_fruit_slash)
```
