## 3. SSRF Korumasi

```
- localhost → blocked
- 127.0.0.1 → blocked
- 0.0.0.0 → blocked
- [::1] → blocked
- 192.168.* → blocked
- 10.* → blocked
- 172.16.* → blocked
- file:// → blocked
- ftp:// → URL parse fail → blocked
```
