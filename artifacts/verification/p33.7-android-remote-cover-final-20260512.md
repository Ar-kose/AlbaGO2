## 2. Cover Zinciri (Android)

```
GET /v1/game-definitions/active?appVersion=0.2.0
  -> JSON response: items[].assets.cover
  -> SupabaseData.fromBackendItems()

  -> AlbaMotionController.supabaseGameToDefinition()
    -> game.metadata["assets"]["cover"] -> AssetManifest.cover
  -> GameDefinition.assets.cover

ShowcaseScreens:
  SportMode -> GameCardUi(imageUrl = game.assets.cover)
  EducationMode -> LearningListCard(imageUrl = game.assets.cover)
  DemoCatalog -> DemoGameCard(imageUrl = game.assets.cover)

  HorizontalGameCards -> AsyncImage(model = card.imageUrl) | fallback painterResource

  DemoListCard -> AsyncImage(model = imageUrl) | fallback painterResource
```
