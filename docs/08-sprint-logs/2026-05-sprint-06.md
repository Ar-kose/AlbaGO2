# Sprint 06 — Beta Readiness & Release Hardening

**Tarih:** 2026-05-08  
**Branch:** `main`  
**Önceki sprint:** [Sprint 05](2026-05-sprint-05.md) — Post-RC Handoff

## Goal

Platform v2 RC'yi "demo edilebilir MVP"den "güvenilir, ölçülebilir, tekrar üretilebilir beta adayına" taşımak.

## Scope

- Release checklist güncellemesi (P6 beta readiness gate)
- Gerçek cihaz manuel katalog yürüyüşü tamamlama
- En az 1 completed game session backend'e kalıcı yazma kanıtı
- Admin publish/refresh akışını tekrar doğrulama
- Risk register ve final beta readiness summary oluşturma
- Known issues güncelleme

## Non-goals

- Yeni native oyun template'i
- Generic scene/action interpreter v2
- Workout session persistence
- Leaderboard production ranking
- Push notification
- Account/profile sistemi
- App Store / Play Store submission
- Büyük UI redesign
- npm audit fix (P4'te accepted non-blocking olarak sınıflandı)

## Known Risks

| Risk | Etki | Mitigasyon | Durum |
|------|------|------------|-------|
| Gerçek cihazda `adb shell input tap` engelli | Manuel otomasyon eksik kalır | scrcpy / fiziksel dokunma / debug direct launch extras | Known |
| Gradle non-ASCII path problemi | Android test worker fail | `GRADLE_USER_HOME=C:\gradle-cache` zorunlu | Mitigated |
| Backend offline | Android katalog boş veya sync fail | debug fallback + sync retry + result preservation | Mitigated |
| Admin invalid game publish | Android parse crash riski | backend publish validation + Android safe parser | Mitigated |
| Session duplicate submit | Skor/leaderboard bozulur | `clientSessionId` idempotency testi | Mitigated |
| Asset format/sanitization açığı | Güvenlik riski | PNG/WebP/SVG allowlist + sanitized SVG + size limit | Mitigated |
