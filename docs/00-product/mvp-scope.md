# MVP Scope

## Dahil

- Android native uygulama
- `SQUAT`, `JUMPING_JACK`, `JUMP_ROPE` motion desteği
- `TARGET_HIT` ve `ENDLESS_RUNNER` template'leri
- Workout session yönetimi
- Remote game definition fetch ve local validation
- Temel leaderboard, daily goal ve reward claim akışları
- İçerik odaklı admin panel

## Hariç

- iOS istemcisi
- Üyelik/ödeme duvarı
- Reklam mediation
- Gerçek zamanlı multiplayer
- Arbitrary script tabanlı oyun motoru

## Başarı ölçütleri

- Güvenilir rep sayımı
- 15+ FPS canlı pose pipeline hedefi
- Oyun varyasyonlarının uygulama güncellemesi olmadan yayınlanması
- Sprint sonunda güncel dokümantasyon
