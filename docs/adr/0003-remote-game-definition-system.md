# ADR-0003: Remote Game Definition System

## Durum

Kabul edildi.

## Bağlam

Yeni oyun varyasyonlarının uygulama güncellemesi olmadan yayınlanması gerekiyor.

## Karar

Uygulama içinde sınırlı template/runtime bulunacak; oyun varyasyonları uzaktan gelen doğrulanmış config ile beslenecek.

## Sonuçlar

İçerik ekleme hızı artar. Yeni template veya arbitrary kod desteği halen uygulama sürümü gerektirir.
